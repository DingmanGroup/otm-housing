How to publish on SharePoint

Embed this interactive page directly into your company SharePoint so colleagues click once and it opens live in their browser.

Part 1 — Upload the file to SharePoint
1
Go to your SharePoint site
Navigate to your Communication site or Company Wide site. Click Documents in the left nav to open the document library.
2
Upload the HTML file
Click Upload → Files and select OTM_Housing_Coordination.html from your Downloads folder. Wait for it to appear in the library.
Keep the filename exactly as-is.
Part 2 — Create the page and embed
3
Create a new SharePoint page
  1. Click + New → Page in the top toolbar
  2. Choose Blank template → Create page
  3. Name it OTM Housing Coordination Fields
4
Add the Embed web part
  1. Click the Toolbox icon (right side panel) → See all web parts
  2. Search for and click Embed
  3. In the Embed panel on the right, paste this URL into the address box:
    https://dingmangroup.github.io/otm-housing/OTM_Housing_Coordination.html
  4. Click Apply — the interactive page will render live
Use the URL method, not the code paste method. SharePoint accepts iframe URLs directly.
Part 3 — Publish and share
5
Publish the page
Click Publish in the top right. The page goes live instantly for everyone in your org.
Use Preview first to confirm the embed renders correctly.
6
Share the link
Copy the SharePoint page URL from your browser and send it via Teams, email, or Slack. Colleagues click once — no download required.
The GitHub URL above also works as a standalone link you can share directly.
