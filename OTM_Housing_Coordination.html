How to publish this page on SharePoint

Follow these steps to embed this interactive file directly into your company SharePoint site so colleagues can click and open it without downloading anything.

Part 1 — Save the file
1
Download this file
Click the download button at the top of this page (or use the link shared with you by Chris) to save the file to your computer.
The file is named OTM_Housing_Coordination.html — keep that filename as-is.
2
Open SharePoint
Go to your company SharePoint site. In the left navigation or top bar, find Site Contents or navigate to the Documents library where your team stores internal resources.
Ask your IT admin if you're not sure which SharePoint site to use.
Part 2 — Create the SharePoint page
3
Create a new SharePoint page
  1. In your SharePoint site, click + New in the top toolbar
  2. Select Page from the dropdown
  3. Choose a blank template and click Create page
  4. Give the page a name — e.g. OTM Housing Coordination Fields
4
Add the Embed web part
  1. On the new page, click the + button to add a web part
  2. Search for and select Embed
  3. In the Embed panel that opens, click Enter code (not a URL)
  4. Open the HTML file you downloaded in a text editor (right-click → Open with → Notepad or TextEdit)
  5. Select all the content (Ctrl+A / Cmd+A) and copy it
  6. Paste it into the Embed code box in SharePoint
  7. Click Save on the Embed panel
The mockup will render live on the page immediately. You'll see the tabs and both field layouts appear in the preview.
Part 3 — Publish and share
5
Publish the page
Click Publish in the top right corner of the SharePoint page editor. This makes the page live and accessible to anyone you share it with.
Use Save as draft first if you want to preview before publishing.
6
Copy and share the link
Once published, copy the page URL from your browser address bar. Share it via email, Teams, or Slack with your colleagues.
Anyone at your company with the link can click it and the page opens directly in their browser — no download required.
Part 4 — If colleagues can't open it
7
Check permissions and custom scripts
If the page shows a blank embed or blocked content notice, your IT admin needs to enable one setting:
  1. Go to SharePoint Admin Center (admin.microsoft.com → SharePoint)
  2. Click Settings in the left nav
  3. Find Custom Script and set it to Allow users to run custom script on personal sites
  4. This may take up to 24 hours to take effect
If you don't have admin access, forward this step to your IT department with the subject line: "Enable custom script for SharePoint Embed web part."
